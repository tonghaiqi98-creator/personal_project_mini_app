# 功能范围

## 本期要做的功能

### 顾客端

- 扫桌号二维码进入小程序。
- 自动识别门店和桌号。
- 查看菜品分类。
- 查看菜品列表和菜品详情信息。
- 将菜品加入购物车。
- 在购物车中调整数量或移除菜品。
- 提交订单。
- 完成模拟支付。
- 查看当前订单状态。

### 商家端

- 查看今日订单列表。
- 按订单状态区分待接单、制作中、已完成订单。
- 查看订单详情，包括桌号、菜品、数量、金额、备注。
- 接单。
- 标记订单完成。
- 查看今日统计数据，包括订单数和销售额。

### 基础能力

- 使用 CloudBase 作为后端服务。
- 使用模拟支付流程，不接真实微信支付。
- 使用固定单门店数据。
- 使用基础数据集合支撑菜单、桌号、订单和统计。

## 本期不做的功能

- 不做外卖配送。
- 不做会员体系。
- 不做积分、优惠券、满减、秒杀等营销功能。
- 不做库存扣减。
- 不做后厨打印机。
- 不做多门店。
- 不做员工权限分级。
- 不接入真实微信支付。
- 不做退款。
- 不做复杂报表。
- 不做客户评价。

## 后续可扩展功能

- 接入真实微信支付。
- 增加门店后台管理菜单。
- 增加菜品上下架和价格维护。
- 增加多门店配置。
- 增加打印机或后厨看板。
- 增加会员、优惠券和储值。
- 增加订单退款和售后流程。
- 增加更完整的经营报表。
